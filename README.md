# PyPlanner
python commandline tool for saving events privately

what is working v0.0.5-beta:
- add a new event
- add a new event to specific calendar
- search by string
- search by date
- view calendars
- view events from specific calendar
- delete calendar
- delete event
- show year/month calendar
- help
